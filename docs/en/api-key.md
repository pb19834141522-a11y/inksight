# Configure API Key

InkSight uses external LLM providers.  
You need a valid API key to generate AI content.

## Supported Providers

- DeepSeek
- Alibaba Bailian
- Moonshot (Kimi)

## Setup

1. Open web config page
2. Choose provider
3. Enter API key and save
4. Trigger refresh to validate output

If generation fails, check provider status and key quota.
